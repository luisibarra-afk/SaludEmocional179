# 🎒 Mochila del Alumno - Bienestar Socioemocional

Aplicación web para el bienestar socioemocional de estudiantes.

## Requisitos

- Node.js 18+
- npm

## Cómo ejecutar

### 1. Backend (API)

```bash
cd backend
npm install
npm run dev
```

El servidor queda en: `http://localhost:5000`

### 2. Frontend (en otra terminal)

```bash
cd frontend
npm install
npm run dev
```

La app queda en: `http://localhost:5173`
